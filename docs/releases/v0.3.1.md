# OpenGrove v0.3.1

OpenGrove v0.3.1 is a sanitizing patch for the v0.3 App release. It removes private App example identifiers from the npm package and keeps the new App employee behavior generic.

## Highlights

- Removed private mounted-App example identifiers from runtime code, harness fixtures, public docs, UI placeholders, and built web/server artifacts.
- Replaced hard-coded App employee detection with generic manifest-declared employees.
- Added App spec documentation for `employees`, `rooms.employees`, and `capabilities.employees`.
- Updated harness coverage to use neutral sample Apps instead of private workflow names.
- Prepared this patch as the clean replacement package after unpublishing the original `0.3.0` npm artifact.

## Verification

Passed before release:

- `npm run typecheck:web`
- `npm run test:harness`
- `npm run check:secrets`
- `npm pack --dry-run`
- package/content scan for removed private App identifiers

# OpenGrove v0.3.1 中文说明

- 测试用例改用中性的 sample Apps，不再引用私有工作流名字。
- 作为干净替换包发布；原 `0.3.0` npm artifact 会先 unpublish。
