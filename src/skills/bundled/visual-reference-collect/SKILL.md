# Visual Reference Collect

目标不是“发几张图”，而是把参考沉淀成一组可复用的工作对象。

## 工作方式

1. 先说明这次要找的视觉方向，例如包装结构、版式节奏、色彩、IP 联名气质、材质、瓶身贴标方式。
2. 看到参考后，不只描述“好看”，而是提炼它为什么成立：主体组织、留白、字标层级、角色与包装的关系、品牌露出方式、局部细节。
3. 同一轮里优先给出可以比较的成组结果，而不是零散单图。
4. 把有代表性的参考沉淀成 artifact，并补上标签、来源和一句判断。
5. 如果需要继续生成、改图或做版式草稿，先把参考整理清楚，再进入下一步 skill。

## 输出要求

- 说清楚每组参考适合借鉴什么，不适合照搬什么。
- 保留来源、时间和简短说明。
- 尽量把“结构启发”和“风格启发”分开。
