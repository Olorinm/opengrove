# OpenGrove Brand Assets

## Visual System

- `opengrove-visual-system.md`: complete visual direction and implementation rules
- `opengrove-style-board.svg`: visual style board
- `opengrove-style-board.png`: PNG export of the style board

## Wordmark

- `opengrove-wordmark.svg`: transparent vector source
- `opengrove-wordmark.png`: transparent PNG export, 640 x 160
- `opengrove-wordmark-on-light.svg`: warm white preview source
- `opengrove-wordmark-on-light.png`: warm white PNG preview, 640 x 160

## Sapling Mark

- `opengrove-sapling.svg`: transparent pixel-vector source
- `opengrove-sapling.png`: transparent PNG export, 128 x 128
- `opengrove-sapling-on-light.svg`: warm white preview source
- `opengrove-sapling-on-light.png`: warm white PNG preview, 256 x 256

## Colors

- Graphite: `#111827`
- Grove green: `#168A53`
- Sapling green: `#5FB24A`
- Sapling highlight: `#7BCB57`
- Sapling shade: `#43A343`
- Sapling trunk: `#202424`
- Warm white preview: `#FBFBFA`

## Typography

The wordmark uses a system sans stack with Inter as the preferred face:

`Inter, ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, SF Pro Display, Segoe UI, sans-serif`
